# EDA-Customers-Products
# Exploratory Data Analysis on Customers & Products

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Customers and Products datasets to understand customer behavior, product trends, discounts, and purchasing patterns using Python.

## Tasks Performed
1. Calculated total inventory value
2. Calculated discounted price for each product
3. Counted total products
4. Listed customers from specific cities/countries
5. Identified most purchased product
6. Visualized top 5 cities with most purchases

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
- `Customers.csv`
- `Products.csv`

## Insights
- Most purchased product: P047
- Top cities with purchases visualized
- Discounted prices calculated
